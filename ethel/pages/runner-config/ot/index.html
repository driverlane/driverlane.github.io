OpenText runners

Current runners created for OpenText products.

Archive Center:

  • OTACStaging - extract Archive Center files using Archive Link.

Content Server:

  • OTCSExtractor - extract Content Server items.
  • OTCSStaging - extract Content Server documents.
  • OTCSLoader - load or update Content Server items.

Core For Success Factors:

  • CoreSFLoader - load files into Core for Success Factors.

Related runners:

  • FSToOTCSTransformer - file sytem objects transformed for the Content Server loader.
  • OTCSToFSTransformer - Content Server objects transformed for the file system loader.
  • OTCSToSPTransformer - Content Server objects transformed for the SharePoint loader.
  • SPToOTCSTransformer - SharePoint objects transformed for the Content Server loader.
