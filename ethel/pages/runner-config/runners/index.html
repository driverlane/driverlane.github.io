Runner types

Runner configuration options can be either specific to the particular runner, or general configuration options used by all or many runners.

Common options are detailed here.

Runner specific configuration details are grouped under the following types:

  • Databases - runners for reading from, and writing to databases
  • Email - runners for working with email files
  • File systems - runners for reading from, and writing to file systems
  • Microsoft - runners for Microsoft products
  • Mock runners - used in testing and demonstrations
  • OpenText - runners for OpenText products
  • Text files - runners for reading from, and writing to text files
