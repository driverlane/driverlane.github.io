Runner status

The various runner status icons represent the following:
Deprecated Not recommended for use. An alternative approach is preferred.
Demo only For testing and demonstration use only.
Planned Planned runner, no work has been undertaken.
Developed An initial version of the runner has been completed recently. It may need more testing and be missing features.
Tested The runner has been developed and tested extensively, but not used in a production job. It may be missing features.
Production The runner has been used previously in a production job.
Veteran The runner has been used in multiple production jobs.
